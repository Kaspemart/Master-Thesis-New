## All Decisions Made
- Model: discrete-time SV, r_t = exp(h_t/2)*ε_t, h_t = μ + φ(h_{t-1}−μ) + σ_η*η_t
- h_0: drawn from stationary distribution N(μ, σ_η²/(1−φ²))
- Interface: batched — simulate_sv(N, T, params, seed); SVParams dataclass; draw_parameters() separate
- Output: returns (N,T), params (N,3), latent_h (N,T) — all three returned and saved
- Storage: .npz format in data/generated/ (git-ignored)
- Parameter ranges: μ ∈ [−10,0], φ ∈ [0.5,0.999], σ_η ∈ [0.05,1.0]
- Series lengths in scope: T = 500, 1000, 2000 (same simulator, different T argument)
- Numerical stability: clip h_t with logged warning
- Seed: optional argument; reproducibility guaranteed when provided
- Implementation style: Balanced — SVParams dataclass + draw_parameters() + simulate_sv()
- Dependencies: NumPy only in simulator (no PyTorch); tqdm for progress bar
- MCMC benchmark (separate feature): PyMC confirmed
