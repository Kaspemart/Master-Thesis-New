## Key Implementation Decisions
- AR model: `rho=[mu*(1-phi), phi], constant=False` — passes intercept explicitly, cleaner than constant=True
- `cores=1` in pm.sample() — mandatory when using joblib multiprocessing (prevents nested subprocess spawning)
- Fast test config: `draws=25, tune=50, chains=2` — chains=2 required for valid R-hat (chains=1 returns NaN)
- Checkpointing: atomic rename (write to .tmp.npz, then rename) avoids corrupted files on crash
